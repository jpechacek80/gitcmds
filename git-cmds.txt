//**help
$ git help			
$ git help config //help on "config options"
$ git help add	//help on "add" cmd

//**global configuration
$ git config --global user.name "jpechacek"
$ git config --global user.email "jpechacek@yahoo.com"
$ git config --list			//shows config settings

//** Create .git directory in current directory 		
$ git init					//creates .git repository dir (or reinitializes)
$ rm -rf .git				//ends version ctrl

//** .gitignore file in current directory
$ //contains a list of excluded files
.project	//ignored folder
*.obj		//ignore

//** stage files for commit
$ git add -A	//add all files/dir for commit
$ git add *.cs	//add specific files
$ git status	// shows commit ready (green)
$ git reset fname  //remove fname from commit ready 
$ git reset	//remove everything staged
$ git status	//Lists Untracked file=readme (red)
$ git status	//Lists modified files (red)

//** sample commit 
$ git add -A	//add all
$ git status	// shows commit ready
$ git commit -m "Ver07"  //add message
$ git status	//nothing to commit(.git is clean)
$ git log		//show hash for commit
If you modify a file after you run git add, 
re-run $ git add again 
to stage the latest version of the file
//** sample add & commit 
$ git commit -a -m "Ver07"
